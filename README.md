## ✨ Main Features

### 🌟 Option 1 — Automatic Pixel Art
Upload an image and instantly convert it into a clean pixel-art grid.

### 🧱 Option 2 — Custom Block Converter
Upload your own block textures. The app matches each pixel to the closest block color and creates a block-based mosaic.

### 🛠️ Option 3 — Manual World Planner
Create a grid and design manually with uploaded blocks.

Includes:
- **Eraser** tool
- **Color Picker** tool
- **Undo** button
- **Stack Blocks** mode
- Background reference image support
- Mouse wheel zoom
- Right-click drag to move around the design
- Fullscreen workspace

### 🔍 Option 4 — Project Inspector
Load a saved `.json` project and click cells to inspect which block was used.

---

## 🧰 Planner Tools

| Tool | What it does |
|---|---|
| 🖌️ Draw | Select any uploaded block and place it on the grid. |
| 🧹 Eraser | Remove blocks from the grid. Dragging works only with Eraser selected. |
| 💧 Color Picker | Pick a block from the canvas and continue drawing with it. |
| ↩ Undo | Revert your last planner action. |
| 📚 Stack Blocks | When enabled, multiple blocks can be placed on the same cell. |

When **Stack Blocks** is off, clicking the same placed block again removes it. Different blocks still cannot replace an existing block unless it is erased first.

---

## 🖥️ Fullscreen Designer

Manual Planner includes a fullscreen design mode for large projects.

In fullscreen:
- The canvas fits the available workspace.
- Use the mouse wheel to zoom in or out.
- Hold right click and drag to move around.
- The uploaded block panel appears on the right.
- Blocks are grouped into **All**, **Blocks**, **BG**, and **Props** categories.
- The block panel shows 20 blocks per page and can be hidden or shown again.

---

## 📦 Inventory & Export

- Uploaded blocks can be removed from the palette anytime.
- The material report updates automatically.
- Export the final design as `.png`.
- Save the editable layout as `.json` and inspect it later.
- Output image width can be set manually; height is calculated automatically.

---

## 🚀 How to Use

1. Choose an operation mode.
2. Upload your image or block textures.
3. Set the grid width and height.
4. Enter the output image width if needed.
5. Generate pixel art or create a planner grid.
6. Edit, inspect, and export your design.
