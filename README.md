# Pixel-Art-converter

🎨 Pixel Art & Block Converter
A sleek, client-side web application designed to convert any standard image into pixel art or a block-based mosaic. This tool is perfect for creators, builders, and gamers (e.g., Minecraft, Growtopia, Pixel Worlds, or bead art) who want to blueprint their designs using custom textures and get an exact materials list.

🚀 Features
Option 1: Automatic Pixel Art: Instantly downscales your image into a classic pixelated grid.

Option 2: Custom Block Converter: Upload your own block textures or palettes. The algorithm analyzes the colors and maps your custom blocks to the closest matching pixels of the main image.

Dynamic Inventory Manager: Manage your uploaded assets on the fly. Don't want a specific block to be used? Simply click the '×' (Remove) button on its preview tile to exclude it.

Real-Time Material Report: Generates a clean, sorted inventory breakdown right beneath the canvas, showing the exact block quantities needed (ordered from most used to least used).

Lightweight & Fast: Runs entirely in the browser using HTML5 Canvas—no backend required.

🛠️ How to Use
Upload the Main Image: Select the target image you want to convert in the first field.

Upload Block Images: Select multiple block/palette textures at once in the second field.

Manage Assets (Optional): If you change your mind about a specific block, hover over its preview tile and click the red × to remove it from the conversion engine.

Select Operation Mode: Choose between Option 1 (Automatic Pixel Art) or Option 2 (Create with Uploaded Blocks).

Set Grid Dimensions: Define the grid size by adjusting the Width and Height (measured in block counts).

Generate: Click the "Generate Pixel Art" button to render your creation and instantly view your dynamic material breakdown below the canvas.
